## 1. analyzers/rank_analyzer.py — 등락률 순위 분석

## 2. analyzers/theme_analyzer.py — 뉴스 기반 테마 자동 분류

### 핵심 원칙
별도 테마 분류 알고리즘을 만들지 않음.
**'이 종목이 왜 올랐는지'는 이미 뉴스에 다 나와 있다'**는 점을 활용.

### 데이터 흐름
```
1. rank_analyzer의 상승 TOP 15 종목 + 각 종목의 뉴스 제목 수집
2. Claude API에 전달:
   "아래 종목들이 오늘 상승했고 관련 뉴스는 다음과 같다.
    같은 이유로 올라간 종목끼리 테마별로 묶고,
    각 테마에 이름을 붙이고 한 줄 설명을 해줘.
    투자 조언은 하지 말고 팩트만 정리해줘."
3. Claude가 테마별 묶음 반환
4. 각 테마에 대표 뉴스 링크 1~2개 배치
```

### Claude API 호출 프롬프트 예시
```
당신은 한국 주식시장 뉴스 큐레이터입니다.
아래는 오늘 상승한 종목들과 관련 뉴스입니다.

[종목 리스트 + 뉴스 제목]

작업:
1. 같은 이유(뉴스)로 상승한 종목끼리 묶어서 테마를 만들어주세요.
2. 각 테마에 이름을 붙여주세요 (예: "원전 관련주", "AI 반도체")
3. 각 테마별로 한 줄 설명을 작성해주세요.
4. 투자 조언/추천은 절대 하지 마세요. 팩트만 정리해주세요.
5. '올랐다'는 쓰되 '오를 것이다'는 절대 쓰지 마세요.

JSON 형식으로 응답:
{"themes": [{"name": "테마명", "description": "한줄설명", "stocks": ["종목명1", "종목명2"], "avg_change": 5.2}]}
```

### 출력
```python
{
  "themes": [
    {
      "name": "원전 관련주",
      "description": "정부의 체코 원전 후속 계약 추진 소식에 원전 관련 종목 동반 상승",
      "stocks": [
        {"code": "034020", "name": "두산에너빌리티", "change_rate": 8.3},
        {"code": "052690", "name": "한전기술", "change_rate": 5.1},
      ],
      "avg_change": 6.2,
      "news": [
        {"title": "정부, 체코 원전 후속 계약 10조 본격 추진", "url": "...", "press": "한경"},
      ]
    },
    ...
  ]
}
```

## 5. generators/blog_generator.py — Claude API 블로그 글 생성

### 기능
- 모든 분석 데이터를 종합하여 Claude API로 최종 블로그 글 생성
- 아침 브리핑 / 마감 리포트 두 가지 템플릿

### Claude API 호출 구성 (총 5~6회)

| 호출 | 역할 | 입력 |
|------|------|------|
| 1 | 상승/하락 해설 | rank_analyzer 출력 + 뉴스 |
| 2 | 테마 분류 | theme_analyzer가 Claude 직접 호출 |
| 3 | 공시 요약 | disclosure_analyzer 출력 |
| 4 | 최종 통합 | 모든 섹션 통합 → 완성된 블로그 글 |
| 5 | 텔레그램 요약 | 전체 리포트 → 5줄 요약 |

### 아침 브리핑 (morning) 템플릿 섹션
1. 미국장 요약 (지수 종가/등락)
2. 미국 주요 종목 (관심종목 등락률)
3. 미국/정치 이슈 (뉴스 요약 + **뉴스링크**)
4. 환율/유가/금
5. 오늘 한국장 주요 일정
6. 면책 문구

### 마감 리포트 (closing) 템플릿 섹션
1. 시장 요약 (KOSPI/KOSDAQ)
2. 상승 TOP 15 (뉴스기반 사유 + **뉴스링크**)
3. 하락 TOP 15 (뉴스기반 사유 + **뉴스링크**)
4. 오늘의 테마 (자동분류 + **대표뉴스링크**)
5. 주요 공시 & 이슈
6. 외국인/기관 수급
7. 면책 문구

### Claude API 최종 통합 프롬프트 핵심 규칙
```
- 투자 조언/추천 절대 금지
- '올랐다'는 쓰되 '오를 것이다'는 절대 안 씀
- 팩트 기반 객관적 서술
- 뉴스 링크는 [제목(언론사)](URL) 형식으로 포함
- SEO: 제목에 날짜 + 키워드 포함 (예: "2026년 3월 18일 주식 상승 종목 정리")
- HTML 형식으로 출력 (Next.js에서 바로 렌더링)
- 면책 문구 하단 필수 포함
```

## 6. generators/telegram_generator.py — 텔레그램 요약 생성

### 기능
- 블로그 글 전체를 5줄 이내로 압축
- 텔레그램 메시지 포맷에 맞게 변환

### 출력 예시 (아침)
```
🇺🇸 3/18 미국장 마감
S&P500 5,638 (-0.7%) | 나스닥 17,844 (-1.2%)
테슬라 -2.1% | 엔비디아 +1.3%
USD/KRW 1,342원 | WTI $78.2

📌 트럼프, 중국산 반도체 추가관세 예고

전체 리포트 → {blog_url}
```

### 출력 예시 (마감)
```
📊 3/18 장 마감
KOSPI 2,847 (+1.2%) | KOSDAQ 892 (-0.3%)
🔥 상승1위 두산에너빌리티 +8.3% (원전 수출)
📉 하락1위 에코프로비엠 -4.2% (EU 보조금 축소)
외국인 +2,340억 | 기관 -1,200억

전체 리포트 → {blog_url}
```
